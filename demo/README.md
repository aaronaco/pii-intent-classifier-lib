# PII Intent Classifier Demo

This space demonstrates the `pii-intent-classifier-lib`, a Python library that uses the Roblox PII Classifier model to detect PII-related intent in text.

## Setup Note
This demo requires access to the gated model [Roblox/roblox-pii-classifier](https://huggingface.co/Roblox/roblox-pii-classifier). 

To run this Space:
1. Accept the model license on the model page.
2. Add a `HF_TOKEN` secret in the Space settings with a valid Hugging Face access token.

For more details, visit the [GitHub Repository](https://github.com/aaronaco/pii-intent-classifier-lib).
